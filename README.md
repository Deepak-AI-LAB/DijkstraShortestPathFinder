Shortest Path Finder using Dijkstra Algorithm

A professional Java Swing desktop application that demonstrates Dijkstra’s Shortest Path Algorithm using interactive graph visualization, modern GUI design, route highlighting, and dynamic navigation between cities.

Features

- Java Swing GUI
- Dijkstra Algorithm
- Graph Visualization
- Login Screen
- Dashboard Navigation
- Shortest Route Calculation
- Route Highlighting
- Modern UI Design

Technologies Used

- Java
- Java Swing
- Graph Data Structure
- Dijkstra Algorithm

How to Run

1. Open project in VS Code or IntelliJ IDEA
2. Compile all Java files
3. Run MainApp.java

Project Type

Mini Project
